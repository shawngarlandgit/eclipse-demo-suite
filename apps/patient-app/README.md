# Eclipse Patient App

This is a local Vite + React app.

## Run Locally

**Prerequisites:**  Node.js

1. Install dependencies:
   `npm install`
2. Set `VITE_CONVEX_URL` in `.env.local` to your Convex deployment
3. Run the app on `http://localhost:3000/`:
   `npm run dev`
