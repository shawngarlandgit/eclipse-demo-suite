## How to Create a PDF from the One-Pager

### Option 1: Browser Print (Easiest)
1. Open `one-pager.html` in Chrome/Safari
2. Press `Cmd+P` (Mac) or `Ctrl+P` (Windows)
3. Select "Save as PDF" as the destination
4. Set margins to "None" or "Minimum"
5. Enable "Background graphics" checkbox
6. Click Save

### Option 2: From Markdown
1. Open `ONE-PAGER.md` in any markdown editor (Notion, Obsidian, VS Code)
2. Export to PDF from that application

## Tips for Using These Materials

1. **Always personalize** — Generic emails get ignored. Reference something specific about the dispensary.

2. **Test different subjects** — A/B test email subject lines to see what gets opens.

3. **Track responses** — Use a CRM or spreadsheet to track who you've contacted and responses.

4. **Follow up consistently** — Most deals close after 5+ touchpoints.

5. **Update regularly** — Add new features, testimonials, and metrics as they happen.
