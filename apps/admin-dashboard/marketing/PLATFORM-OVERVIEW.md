## Core Features & Capabilities

### 1. 8 Customizable Dashboard Layouts
- 4 Simple views + 4 Detailed views with drag-and-drop widgets
- Real-time KPIs: Revenue (daily/MTD/YTD), transaction counts, inventory health
- Compliance flag monitoring with severity indicators
- Each staff role sees only what they need

### 2. Advanced Inventory Management
- Multi-batch tracking with THC/CBD percentages
- Automated reorder alerts and stock level monitoring
- Metrc integration for seed-to-sale compliance
- CSV bulk import with validation
- Compliance alerts for testing deadlines and inventory variances

### 3. Dual-Mode Order System
- **Budtender Mode**: Full POS with customer lookup, ID scanning, recommendations
- **Kiosk Mode**: Customer self-service with express checkout
- Real-time cart updates with tax calculations
- Payment integration ready (Square, Clover)

### 4. AI-Powered Strain Recommendations
- Proprietary similarity algorithm based on terpene profiles
- Effect-based filtering (relaxation, pain relief, energy, focus, etc.)
- Customer preference learning over time
- Budtender recommendation history tracking

### 5. PDF417 ID Scanner
- Parse any US driver's license barcode instantly
- Auto-extract: name, DOB, address, expiration, medical card status
- Age verification with configurable thresholds
- HIPAA-compliant customer record creation

### 6. 6-Tier Role Hierarchy

| Role | Access Level |
|------|--------------|
| Patient | Own profile, recommendations, questionnaire |
| Budtender | Sales floor, customers, recommendations |
| Staff | Extended inventory views |
| Manager | Full inventory, analytics, staff management |
| Owner | Compliance, integrations, all reports |
| Admin | System-wide access, multi-dispensary |

### 7. Enterprise Security & Compliance
- Row-level security with multi-tenant isolation
- HIPAA-compliant audit logging (every view, edit, delete)
- PII hashing for customer records
- Automatic compliance flag generation
- Metrc sync status monitoring

### 8. Real-Time Everything
- WebSocket subscriptions for instant updates
- Live inventory sync across registers
- Real-time compliance alerts
- Staff activity monitoring
